# :hammer: javascript-weird-parts

**Syntax parser**: A program that reads your code and determines what it does and if its grammer is valid  
**Lexical environment**: Where something sits physically in the code you write  
**Execution context**: A wrapper to help manage the code that is running  
**High Order Functions**: The functions that take a function as an argument, or return a function as a return value are called higher order functions.
